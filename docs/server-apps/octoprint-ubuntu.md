# OctoPrint Ubuntu

Install lubuntu on laptop. 

## pyenv Installation and Setup

**Install PyENV**

In a terminal, run:

```
curl https://pyenv.run | bash
```

**Install Python 3.9.19**

(or latest version of python)

In a terminal, run:

```
pyenv install 3.9.19
```

**Set Installed Python Version as Default**

In a terminal, run:

```
pyenv global 3.9.19
```

## Install OctoPrint

Use octoprint_deploy.sh script to install OctoPrint on Ubuntu. Follow documentation on the [OctoPrint_deploy github repo (github.com/paukstelis/octoprint_deploy)](https://github.com/paukstelis/octoprint_deploy?tab=readme-ov-file#how-to-use)
